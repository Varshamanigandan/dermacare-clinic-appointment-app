# DermaCare Clinic — Appointment Booking App

## Task
Design a mobile app UI for an appointment booking system for a service-based business (clinic, salon, consultant, fitness trainer, etc.).

## Business
A dermatology/skin clinic offering treatments such as acne treatment, consultations, laser therapy, and skin check-ups — designed with a clean, trustworthy medical aesthetic.

## Prototype
🔗 [View the Figma prototype](https://figma.com/make/x3cp1LmuT4mS2QW9eMI1fy/DermaCare-Clinic-Appointment-App)

## Screens
- **Onboarding / Sign In** — Trust signal (Board-Certified Dermatologists) before login
- **Service Selection** — Tappable treatment cards (Acne Treatment, Consultation, Laser Therapy, Skin Check-up)
- **Specialist Selection** — Dermatologist profiles to choose from
- **Date & Time** — Interactive calendar with disabled past dates and time slot grid
- **Confirmation** — Full booking summary with reference number and add-to-calendar action

## Tools Used
Figma (AI-assisted UI generation + interactive prototyping)

## Key UX Decisions
- Step progress indicator across the booking flow to reduce drop-off
- Specialist chosen before time slot, so availability reflects the actual doctor selected
- Already-booked time slots shown (not hidden) for a transparent schedule
- Booking reference number + add-to-calendar to reduce no-shows
